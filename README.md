# Dashboard CSS3

Autor: Brian Oropeza  
Programa: Ingeniería de Sistemas  
Universidad de Santander

## Descripción

Este proyecto consiste en la implementación de un dashboard web utilizando CSS Grid y Flexbox sin el uso de frameworks externos.

## Tecnologías utilizadas

HTML5  
CSS3  
CSS Grid  
Flexbox

## Ejecución

1. Abrir el proyecto en VS Code
2. Instalar extensión Live Server
3. Ejecutar index.html con Live Server
